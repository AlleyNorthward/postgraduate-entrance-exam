&emsp;&emsp;已经决定要考研了. 我的目标是什么呢?<br>
&emsp;&emsp;可能跟大部分人想的不同, 对我而言并不是说考多少分, 考上哪个大学, 是我的目标. 我自己没有目标, 期望的是`8`个月之后能成为什么样的人.<br>
&emsp;&emsp;我期望自己`8`个月后是个什么样的人呢?<br>

- 能够分析各种数学题目, 能够按照自己的想法解决. 在抖音刷到相关数学题目时, 不会划走, 而是主动思考一番, 看看有没有解决方案.
- 能够读懂英语书, 看英语视频的时候不再需要字幕. 未来工作实践时, 中文技术书籍可能已经无法满足我的需求, 阅读英文技术书会成为必然. 我期望英语能够融入我的生活, 不仅仅是为了考试, 更是有一种获取高阶技术的选择. 不论考研成功与否, 我期望以后能养成看英语书, 听英语, 讲英语的习惯.
- 能够明白底层计算机工作的原理. 这是必然的, 学计算机的人不懂, 那么谁会懂呢? 仅靠做题肯定远远不够的. 实践才是更重要的.
- 能够手搓小操作系统, 明白现在操作系统工作原理.
- 能够明白计算机网络交互原理, 深层次理解计算机网络各层存在的意义.
- 能够精通数据结构, 明白内存管理, 知道`stl`底层实现原理, 能做各类算法题(这个很困难, 但是我现在一直这么准备着).

假使考研没成功, 我能完成上述大半部分的话, 我也无悔了.
考研成功进军图形学; 考研失败进军嵌入式. 不论学习什么, 上述的内容都是必定要掌握的.

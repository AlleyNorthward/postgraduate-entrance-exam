# 描述
- 对于算法问题, 我们已经将其分成如下三层
	- 输入
	- 算法
	- 输出
- 而是在内部, 参考坐标系总会存在问题, 所以这里想要专门强调一下
	- **for**循环(不论哪层), 都是**base-0**, 最好都是$\leq offset$, 而不是 $< length$
	- 算法内部实现主体一定是**base-0**, 例如返回值
		- 如果问题是需要转到base-1, 那么就在输出时操作, 而不是交给算法操作

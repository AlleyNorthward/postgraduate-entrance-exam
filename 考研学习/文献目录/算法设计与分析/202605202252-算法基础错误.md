# 描述
- 这里分析的是基础错误, 而不是算法本身严重的理解性错误
- 这里对应的是[[202605202237-输入错误|输入错误]]与**输出错误**, 是**非算法性**错误, 输入基础错误
- 关注点
	- 算法相关函数的**传参问题**
# 基础错误
## 案例1
~~~C
  IO_ *io = (IO_ *)malloc(sizeof(IO_));
  input(io);
  for (int i = 0; i < query_size(io) - 1; ++i) {
    add_output(io, algorithm(io, i));
  }
~~~
- 这里是严重的传参错误, 传入的是`i`, 因为写的时候以为是`i`, 但并不是, 需要思考一下

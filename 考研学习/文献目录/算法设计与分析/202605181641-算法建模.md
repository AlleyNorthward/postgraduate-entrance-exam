# 描述
我们现在要抽象一种结构性思想, 用来对算整体进行建模. 参考思想为[[202605102217-向量空间|向量空间]]
- 状态空间
	- **数据**是状态空间中的元素
	- **数据结构**则是描述状态空间元素的关系
	- **算法**是一种状态映射, 将输入状态空间中的元素与结构映射成输出状态空间中的元素与结构. **子算法**则是将算法进行拆分. 算法跟所有子算法的关系类似**复合**关系, 但并不完全是(除非子算法的输入与输出完整匹配)
	- **函数**是对算法的封装, **子函数**则是对子算法的封装. 同理, 函数跟子函数也有类似的**复合**关系
	- **系统**则是函数空间, **模块**则是对函数空间内部进行的划分
	- **程序**是离散**向量场**, 是从状态空间到切空间的一种映射
	- **每行代码**是**切空间**中的向量$v(s_i)$, 执行代码则是沿着向量场积分, 状态沿轨迹的变化

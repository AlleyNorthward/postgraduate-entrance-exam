# 描述
- 关注点:
	- `sizeof()`加常数, 有无添加括号
	- 实体[[指针]]相关赋值有无问题
	- `for`循环, 按照自己的分析规则吧, 后续就`<= size(arr) - 1`
		- 严格遵守有效的存储空间思想吧
		- 但是也不要太死, 但这确实是关注点
# 空间开辟
## 案例1
~~~C
static void add_output(IO_ *io, int n) {
  io->output = (int *)realloc(io->output, sizeof(int) * output_size(io) + 1);
  io->output = &n;
  ++io->output_count;
}
~~~
- 很明显, `output_size(io) + 1`, 应该添加括号
- 另外, `&n`升维, 最后`io→output`指向了`n`的地址, 肯定错误, 应该是`io→output[output_size(io)] = n`

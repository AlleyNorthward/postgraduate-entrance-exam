- 文法不是再讲字符流了, 而是分析语法层面的了
- 关键概念
	- 非终结符
		- 语句, 计算等
	- 终结符
		- token, 标识符等
	- 产生式
		- 有左部和右部
		- 非终结符一定是左部(可以再分)
		- 终结符一定不会出现在左部
	- 开始符号
		- 一般就是某语句划分的
- L是一种映射关系, 将语法变量映射(非终结符)到了字符串集合(终结符)
	- 例如L(O) = {+,-}
	- L(S(开始符号))就是所给文法定义的语言

# 举例
例如:
~~~C
int a = 10;
~~~
其中int, a, =, 10, ;, 都是终结符, 其本身是赋值语句, 是非终结符
~~~C
int a = b + 10;
~~~
其中, int , a, =, 10, b;, 都是终结符, 赋值语句和表达式, 是非终结符

# 关联
- 所以, 在[[202606101842-基本组成|基本组成]]中, 单词, 就是终结符, 语句是非终结符, 程序是可是符号
- 不不不, 书上说, 语句是句子
- 但是我的逻辑中, 语句是终结符, 例如举例中说的, 是语句, 但是int a = 10; , 本身是句子
	- 代码段是句子, 但是中间推导是句型
# 区分
- 文法中的变量, 是指语法变量, 表示的是非终结符, 而代码中的变量是终结符, 这点需要区分一下.

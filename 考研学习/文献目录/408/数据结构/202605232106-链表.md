# 关键名词
- 头指针 head:
	- 指向第一个真实节点。
- 头结点 / 虚拟头结点 dummy:
	- 一个不存真实数据的假节点,放在链表最前面,方便操作.
- 首元节点:
	- 第一个真正存数据的节点。
- 尾节点:
	- 最后一个真实节点,它的 next 是 0 / NULL.
- 尾指针 tail:
	- 一个变量,专门记录尾节点是谁,方便尾插.
# 描述
- 头指针跟头节点区别:
~~~C
// 头指针
newNode->next = head;
head = newNode;
// 头节点
newNode->next = dummy->next;
dummy->next = newNode;
~~~

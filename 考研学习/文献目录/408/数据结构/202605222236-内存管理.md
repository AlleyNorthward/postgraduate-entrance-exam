#### 下面代码问题在哪?
~~~C
...
void matrix_init(State* state){
	int** matrix = state->matrix;
	matrix = (int**)malloc(sizeof(int *));
	...
}
...
~~~
- matrix是栈上指针, 初始值是state->matrix地址, 后续更改了
- 所以state->matrix仍是野指针
	- 思考一下, 如果返回栈上的堆中开辟空间, 是否有问题吗? 
		- 没有, 会被释放, 但是地址还是会被返回的
- matrix有三行, 上面就只有1w行

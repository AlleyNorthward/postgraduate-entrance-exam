# 背景
- MAR是地址寄存器, MDR是指令寄存器
- 早期计算器的存储字长和机器的指令字长,数据字长相等
- 为了适应指令和数据字长的可变性, 其长度不由存储字长来确定, 而是由字节确定

# 定义
- 1个字节(Byte)被定义为由8位(bit)二进制代码组成
# 描述
- 此时存储字长, 指令字长, 数据字长三者可各不相同, 但它们必须是字节的整数倍

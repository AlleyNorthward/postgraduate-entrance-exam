# 原码空间
$8$位原码
- 加法: $a+b$
- 转换补码
	- $mod \space m$
	- 正数不变, 负数的补码=$m-|$原码$|$
- 有效位仅$7$位, 最高位仅表示正负
# 补码空间
$8$位补码
- 加法$(a+b)mod\space m$
- 转换源码
	- 小于$m/2$不变, 高于$m/2$则$|$源码$|$=$m-$补码
	- 最高位当做$-128$, 然后整体相加即可
- 有效位是$8$位
# 举例
例如
- $4-2$和$2-4$
- 对于$4-2$
	- 都是属于原码空间, 映射到补码空间
	- $4$是$4$, $-2$是$256-2=254$
	- 则补码空间中, $(4+254)mod256 = 2$
	- 这个$2$是补码空间, 需要转换到原码空间, 仍为$2$
	- 再用二进制分析一下
		- $4$是$00000100$, $2$是$00000010$
		- $4$的补码不变, $2$的补码是取反$+1$, 即$11111110$
		- 然后补码相加, 即$(00000100 + 11111110)mod2^{8}$
		- 即$1 00000010$, 舍弃最高位后, 为$00000010$, 也就是$2$
		- 这里的$2$是补码, 转换成原码就是$2$
- 对于$2-4$
	- $2$的补码是$00000010$, $-4$的补码是$11111100$
	- 补码相加后为$11111110$
	- 此时补码是$254$, 需要转换成原码
	- 两种方式, 一种是直接计算, 它是负数, 所以原码绝对值是$2$, 原码是$-2$
	- 另一种是最高位视为$-128$, 整体相加后也能得到$-2$
	- 这也是补码空间到原码空间的一种映射

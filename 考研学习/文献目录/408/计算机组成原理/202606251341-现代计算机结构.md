- 在[[202606251339-冯诺依曼计算机结构|冯诺依曼]]计算机结构之上, 现代计算机将运算器(可以认为是ALU)和控制器(可以认为是CU)统称为中央处理器**CPU**, 输入设备和输出设备简称为**I/O设备**
- 这样, 现代计算机可认为由三大部分组成: 
	- CPU
	- I/O设备
	- 主存储器(Main Memory MM)
- CPU与主存储器称为**主机**, I/O设备称为外部设备

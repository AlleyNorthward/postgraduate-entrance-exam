# 组成
- 指令由**操作码**和**地址码**组成
	- 操作码用来表示操作的性质
	- 地址码用来表示操作数在存储器中的位置
# 描述
- 流程如下
- RAM(内存)中编写指令和数据
- PC(程序计数器)开始执行
	- 这部分可以看看66那个视频, 或者拿出来自己的板子, 好好看看
- 从RAM取指放到IR(指令寄存器)中
- IR的OP(操作码)+计数器输入到ROM中, 映射位控制信号
- IR的地址通过总线输入到MAR中, 获取相关数据, 通过总线送入对应单元
- 执行

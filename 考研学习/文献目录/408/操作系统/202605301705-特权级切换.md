- 操作系统如何实现内核态到用户态的切换?
- 操作系统如何实现任务间的切换?
- 想要实现, 都离不可特权级切换这个概念
- 下面`42-48`行, 是特权切换的必要操作
~~~C
_start_32:
  mov $KERNEL_DATA_SEG, %ax
  mov %ax, %ds
  mov %ax, %es
  mov %ax, %ss
  mov %ax, %gs
  mov %ax, %fs
  mov $_start, %esp

  call os_init

  mov $pg_dir, %eax
  mov %eax, %cr3

  mov %cr4, %eax
  orl $(1 << 4), %eax
  mov %eax, %cr4

  mov %cr0, %eax
  orl $(1 << 31), %eax
  mov %eax, %cr0

  mov $TASK0_TSS_SEG, %ax
  ltr %ax

  push $APP_DATA_SEG
  push $task0_dpl3_stack + 1024 * 4
  push $0             # $0x202
  push $APP_CODE_SEG
  push $task0_entry

  iret
~~~

- `iret`指令会干什么?
	- 处理中断函数时, 最后会来上这么一句
	- 它会从栈弹出EIP, CS段选择子,弹出EFLAGS
	- 特权切换时, 还会弹出ESP, SS
		- 所以同级别切换, 弹出12字节
		- 跨级别切换, 弹出20字节
	- 注意, 这里的弹出不是人为弹出, 而是指令自动弹出
	- EIP → CS → EFLAGS → ESP → SS 这是弹出顺序, 所以压栈需要反着压
- 因此当我们想要跨级别切换时, 需要向栈内push相关内容
- 注意第`44`行, 当切换到待执行任务时, 需要给其配备任务栈
	- 思考之前实践时, 实例化任务时, 总会开辟栈空间
	- 但是这里呢, 栈空间似乎并不是连续的, 所以可以专门弄个栈段, 用于存放栈空间

# 组成
- 线性泛函(行向量)
- 系数向量(列向量)
- [[202605111201-矩阵|矩阵]]
# 描述
- 为了更好地解释$Ax=b$, 而抽象出来的一种概念
- 主动作用的对象是左对象
# 实例
- 线性泛函(行向量), 作用于[[202605111117-右对象|右对象]]测量向量(列向量)
- 系数向量(列向量), 作用于右对象向量组(列向量矩阵)
- 矩阵(行列视角都可), 作用于向量(视矩阵而定)

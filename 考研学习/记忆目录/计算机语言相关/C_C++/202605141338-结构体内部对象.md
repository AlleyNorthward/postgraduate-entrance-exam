# 组成
- 实体对象
	- 非指针类型
	- 首地址指针
- 指针对象
	- 指向实体对象的指针
# 描述
- 结构体中, 除了基本数据类型外, 抽象数据类型大概就上述这几种
- 非指针类型实体对象, 一般就是真实实体, 声明周期由本层结构体维护
- 首地址指针实体对象, 一般是某数据结构, 也是真实实体
- 实体对象可以在堆上开辟空间
- 指针对象呢, 则是指向实体对象的, 声明周期并不不由本层结构体维护
- 对于指针对象会存在两个结构体互相存有对方的指针, 这是没有任何问题的, 并不会存在循环引用的问题
- 但是存有对方的实体对象的话, 逻辑上是存在问题的(从面向对象角度很容易分析)

C语言中, 结构体声明有两种方案, 结构体实现有四种方案, 我们都来看一下.
# 描述
- 声明1
~~~C
struct A; //提前声明, 但是引用时必须携带struct, 且只能用指针指向

struct B{
	struct A* pa; // 合法
};
~~~
- 声明2
~~~C
typedef struct A A;

struct B{
	A* pa;
};

// 这个方法就简便了, 但是一开始的typedef还是容易让人困惑的
~~~

- 实现1
~~~C
struct A{
	...
};

struct A a;
// 最普通方案
~~~
- 实现2
~~~C
struct A{
	...
}s1,s2;
// 这是实现+定义变量
~~~
- 实现3
~~~C
typedef struct A{
...
}A;

// 最常用, 最干净
~~~

- 实现4
~~~C
typedef struct {
...
}A;
// 不建议使用, 没有名字, 无法向前声明
~~~

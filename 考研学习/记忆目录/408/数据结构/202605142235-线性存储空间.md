# 组成
- base 基地址
- offset 偏移
- length 长度
- ptr 指针
	- idx_ptr 下标指针
	- adr_ptr 地址指针
# 描述
- 这里有[[202605121153-基变换公式|基变换]]的思想, 关键是坐标系选取统一
- 例如, 在同一个数组中, 分析下标指针, 没有问题
- 可是如果有地址指针, 或者下标指针初始位置不为base, 就会存在问题
- 这样需要转换为我们要求的形式, 才能做到相对的统一, 就可以直接分析问题了
# 实例
- [[202605142140-lua源码-栈相关#请自己写一下lua栈中的lua_settop()函数吧.|实例1]]
	- 我觉得这个实例还是十分典型的
	- 关键点为什么idx是base-0的, 也就是idx为什么跟offset等价
	- idx=0时, 是在base上方+1处, 但是idx跟top等价, top指向地方为空, 所以idx=0时, 上方也是空的, 此时偏移为0, 如此判断idx跟offset等价, 于是idx_ptr=idx

#### 线性存储空间内部的对象关系是什么? 提示: offset两种种表示方式, max_offset两种, adr_ptr跟idx_ptr的转换
- offset = idx_ptr
- offset = adr_ptr - base
- max_offset = 有效的end - 有效的start
- max_offset = length - 1
- adr_ptr = idx_ptr+ base
<!--ID: 1778827687895-->

#### 对于给定的ptr, 如何映射成自己的adr_ptr后者idx_ptr?
- 对于idx_ptr, 分析取较小数字时跟offset的差距. **注意**取有效的下标
- 对于adr_ptr则直接判断跟base关系即可

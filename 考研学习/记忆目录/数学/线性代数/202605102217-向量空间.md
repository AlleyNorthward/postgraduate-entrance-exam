#### 用抽象分析方法分析一下向量空间
- 抽象对象分析
	- 本身对象是个集合
	- 输入对象无
	- 应用对象无
	- 输出对象无
- 抽象记号分析
	- 是个集合, 但是内部元素异常抽象, 无法表示
- 抽象空间分析
	- 思考其内部元素, 向量
	- 当然, 若为函数空间, 元素就是函数了

#### 向量空间中, 经常会有加性, 齐性分析. 最本质的分析策略是什么?
- 给定一个向量, 先分析它是什么
- 然后向上找其算子
- 那么就会有$D(u+v) = Du + Dv$,$D(\lambda v) = \lambda D(v)$

例如:
- 线性映射, 向上可以找算子$D$, 向下可以作为算子作用于向量, 因此有两种写法
- $D(S+T) = DS + DT$以及$D(\lambda S) = \lambda DS$
- $T(u+v) = Tu + Tv$以及$T(\lambda u) = \lambda Tu$

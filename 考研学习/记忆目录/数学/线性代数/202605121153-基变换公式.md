# 描述
例如$T(x,y) = (2x+3y,4x+5y),x=3a,y=4b$求$T(a,b)$
- 有$A=[2\space3,4\space5]$, $P=[3\space0,0\space4]$
- 此时可以求$P^{-1}$
- $P$称作坐标变换矩阵, 在积分中经常用
- 这里的$P$, 是由$x=3a,y=4b$组成的, 所以输入是$[a,b]$
- 经过$P$变换, 就变成了$[x,y]$可以被$A$作用了
- 于是就有$B=P^{-1}AP$
- 输入$[a,b]$,经$P$转换后被$A$作用, 再翻转回去
# Anki
#### 如何理解$P$?
- 坐标变换矩阵, 十分关键

# 描述
- 在不同基下, 总会有$v=xe_x+ye_y = ae_a+be_b$.

# Anki
#### 怎么理解基不变原则? 假定给定向量$v$, 在$xy$坐标系下坐标为$(x,y)$, 在$ab$坐标系下坐标为$(a,b)$
- 不论什么, 总会有$v=xe_x+ye_y=ae_a+be_b$

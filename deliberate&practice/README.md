&emsp;&emsp;这是刻意练习小程序.<br>
&emsp;&emsp;有什么用呢? 强化练习. 虽说无效的重复没有意义, 但是短时内, 都某一知识点进行疯狂提取, 效果可能会超出预期. 比如我曾用其它的一个小程序, 练习`4`位二进制数, 十进制数, `1`位十六进制数的转换(混合练习), 现在能非常快速知道它对应的数字. 比如`c`, 十进制是`12`, 二进制是`1100`.<br>
&emsp;&emsp;所以有时候, 某些知识点不是记不住, 而是提取的不够. 这个程序就是用来协助的. <br>
&emsp;&emsp;比如今天我学了线性代数, 明白了结构, 也能说上来, 但可能会想上一会. 经过这个程序练习后呢, 就能快速提取了.<br> 练习是必须的, 但是选择合适的时间练习更加重要.
&emsp;&emsp;注意, 这个程序是短期练习, 长期练习需要`Anki`卡.<br>
